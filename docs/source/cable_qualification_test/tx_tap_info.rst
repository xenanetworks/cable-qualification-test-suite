TX Equalizers Information
===========================================

The TX Equalizers Information Information collects the TX equalizer values of each lane.

Setup
-----


Method
----------

1. Read and save the TX equalizer values of each TX ports, including:
    
    * Pre3 (dB)
    * Pre2 (dB)
    * Pre (dB)
    * Main (mV)
    * Post (dB)
